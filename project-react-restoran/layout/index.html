Selamat Datang di Aplikasi CRUD dengan Axios

Silakan pilih menu di atas untuk melakukan operasi data:

  • GET - Menampilkan semua data pelanggan
  • SHOW - Menampilkan data pelanggan berdasarkan ID
  • POST - Menambahkan data pelanggan baru
  • DELETE - Menghapus data pelanggan
  • UPDATE - Mengubah data pelanggan
